Formulas


The main function used for this fractal is: f(z)=(z^2)+c, which we can see is an iterative one.

Where c is a constant complex number (a+b*i), and z(n+1) es equal to (z(n)^2)+c, and z(0)= z.

If we develop this last formula, we can see that (a+b*i)*(a+b*i) = (a^2 - b^2) + 2*(a*b*i).

Where we are going to end with z(n+1) = (a^2 - b^2) + 2*(a*b*i) + a + (b*i).
